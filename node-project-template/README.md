## Setup
### NVM 

To use the correct node version run:
```
nvm use
```

If you don't have nvm, [install it first](https://github.com/creationix/nvm).

### Yarn

Install yarn globally if you don't have it:
```
brew install yarn
``` 

Instead of running `npm install` use:
```
yarn install
```
